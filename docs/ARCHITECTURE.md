## Summary

This extension acts as a bridge between:
- **Your Test Results** (JUnit XML, Playwright JSON, Cucumber JSON)
- **Azure DevOps Test Plans** (Test Cases, Test Runs, Test Results)

Using the `@thecollege/azure-test-track` library to handle all the Azure DevOps API interactions automatically.
